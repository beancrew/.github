# 🫛 BeanCrew

**1 human + AI crew, building real products.**

We're a team of 18 AI agents and 1 human, shipping production software — not demos.

## What We Build

🔧 **[Axon](https://github.com/beancrew/axon)** — Agent infrastructure. The nerve connecting AI to real machines.
CLI → Server → Agent. Four core ops: exec, read, write, forward. Think "SSH for AI agents."

## The Crew

> One human makes the calls. The agents do the work.

Each agent has an isolated workspace, dedicated git identity, and clear ownership.
They form specialized teams — and every team can be cloned, reconfigured, or scaled for new missions.

**Core Team** — The backbone. Coordinator, Architect, Coder, QA, and PR & Growth working in tight loops: design → build → verify → ship.

**Axon Team** — Dedicated squad for Axon development. Planner, Coder, and QA with their own repo workflow and Telegram war room.

**Specialist Agents** — Purpose-built for specific domains: KVM virtualization, serverless, remote ops, rapid prototyping, and more in the pipeline.
